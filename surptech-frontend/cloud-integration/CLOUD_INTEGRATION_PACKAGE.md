### For Developers (Quick Setup)

**Install Dependencies** (2 min)
```bash
npm install firebase @google-cloud/storage mongodb
```

**Create Environment File** (3 min)
```bash
# Create .env.local with your credentials
NEXT_PUBLIC_FIREBASE_API_KEY=your_key
NEXT_PUBLIC_FIREBASE_PROJECT_ID=your_project
GCP_PROJECT_ID=your_project
GCP_CREDENTIALS_PATH=./credentials.json
MONGODB_URI=mongodb+srv://...
```

**Start Development** (Ongoing)
```bash
npm run dev
# Follow INTEGRATION_BASELINE.md for code examples
```
